# Check Existing Process Functionality

Before implementing new process logic, search existing processes and subprocesses.

## Arguments
- `$ARGUMENTS`: description of the process functionality needed

## Instructions

1. Search all .proc files in app/diagrams/
2. Search Groovy scripts in app/src-groovy/
3. Search connector configurations in app/connectors-conf/
4. Check the shared extension library for reusable utilities
5. Recommend: reuse subprocess, extract to library, or create new
